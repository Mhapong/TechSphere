### **Prerequisites**

Ensure you have the following installed:

- Node.js (>= 20)
- npm

### **Frontend**

```bash
cd frontend
npm install
```

### **Backend**

```bash
cd ../backend
npm install
```

### **Run the Application**

#### **Start Backend (Strapi API)**

```bash
cd backend
nvm use 20
npm run develop
```

#### **Start Frontend (React App)**

```bash
cd frontend
nvm use 20
npm start
```
